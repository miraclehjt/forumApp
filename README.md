# forumApp
基于JAVA的程序员知识学习类App的设计与实现项目文档

开发环境介绍：
Eclipse、Hbuider、Redis2.0、Mysql5.0、JDK1.7、tomcat7.0

用到相关的技术：
Spring、springmvc、mybatis、MUI、CastApp.js、shiro、redis、easyUI、jquery、javaScript、h5、css、jsp、html

项目运行说明：
后台服务端运行到tomcat中。访问项目时需要启动Redis服务，运行环境在android手机。

项目文件说明：
StudentAppCourse 文件是前台APP端代码实现文件，部署到hbuider工具上
ForumAppService 为服务端代码实现，部署到eclipse中
数据库文件 studentforumapp.sql
